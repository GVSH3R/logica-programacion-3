# Logística de Programación: Cálculo de Factorial

Este repositorio contiene una aplicación web mínima diseñada para resolver el cálculo del factorial de un número entero positivo, desarrollada como parte de los ejercicios de lógica de programación de Generation México.

## Descripción del Proyecto

El objetivo principal es implementar un algoritmo eficiente para calcular el factorial ($n!$) mediante la interacción directa con el usuario a través del DOM. El sistema incluye validaciones para asegurar que la entrada de datos sea un número entero no negativo, cumpliendo con los estándares de robustez de software.

## Características Técnicas

* **HTML5 & CSS3:** Interfaz de usuario limpia con separación clara de estilos y estructura.
* **JavaScript (ES6):** Implementación de lógica iterativa para el cálculo matemático.
* **Validación de Datos:** Control de errores para entradas no numéricas, vacías o negativas.
* **UI/UX:** Eliminación de controles nativos del navegador (*spinners*) para una experiencia visual consistente.
